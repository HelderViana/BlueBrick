How to extend the Part Library

One of the strongest point of BlueBrick is its facility to extend the part library. Basically the Part Library is created when the application start up by parsing the files present on your hard-drive. Adding or removing parts is as simple as adding or removing files in the parts folder. If you want to know how to create and add your own parts to BlueBrick, You should follow these steps:

  1. First take a look on how the parts folder is organized.
  2. Then create a new picture for your part.
  3. Optionnally add a part description file in xml.
  4. And you should read about the "Reload Part Library" feature usefull when you are developping new parts.
  5. Then if you want to add a textual description for your part, you will need to modifiy the Part Info File.
  6. Finally, for a better compatibility of your new part with LDRAW, you may also want to edit the LDRAW remap file.
  7. And the same thing for the Track Designer remap file.
