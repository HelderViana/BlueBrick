Options d'apparence


Afficher les Points de Connections Libres alterne les différents indicateurs de connections entre actif et inactif. Ceux-ci sont actifs par défaut pour visualiser les points libres ainsi que les liens brisés entre les pièces. Vous pouvez mettre cette option à inactive lorsque vous avez complété votre plan.

Par défaut le Schéma de Couleurs est "Blue Print". Vous pouvez sélectionner une version Parchemin ou Classique. Les schémas de couleurs personnalisés peuvent être configurés en sélectionnant "Custom" du menu déroulant et choisissant la couleur en cliquant sur la boite de couleurs à la droite du texte. "Background Color", "Default Grid Color" and "Default sub-grid Color". Le résultat de votre choix de couleurs sera présenté dans la fenêtre de prévisualisation. Ces changements ne seront effectifs que lorsque vous aurez cliqué sur le bouton "Ok" dans la fenêtre Options de BlueBrick.
