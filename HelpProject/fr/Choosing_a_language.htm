Choisir une langue

La première chose dont vous voudrez probablement faire est de changer la langue du programme BlueBrick. Pour effectuer ce changement, vous devez aller à  edit > global options > general  et choisir la langue désirée du menu déroulant dans le coin supérieur gauche. Appuyez sur Ok pour fermer la fenêtre et redémarrer le programme afin que le changement soit effectif.

Si le fichier d'aide est traduit dans la langue de votre choix, la prochaine fois que vous sélectionnerez  help > content and index , l'information d'aide sera affiché dans la langue active de l'application.
