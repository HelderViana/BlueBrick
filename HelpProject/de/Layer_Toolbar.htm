Die Ebenen-Werkzeugleiste


Die Ebenen-Werkzeugleiste unten rechts bietet die wichtigsten Werkzeuge zur Erstellung und Änderung von Ebenen

Ebene verschieben

Durch Anklicken der Pfeile 1 bzw. 2 wird die gewählte und somit grau hinterlegte Ebene (hier Ebene "Track" (8)) um eine Ebene nach hinten bzw. nach vorn verschoben. Im Fensterausschnitt rechts unten werden die Ebenen in der entsprechenden Reihenfolge angezeigt. Hier ist die Textebene "Layout Name" im Vordergrund, dahinter die Flächenschicht "Control Station", dahinter die Steineebene "Track" usw.

Ebene hinzufügen

Mit den Schaltflächen 3 bis 6 werden neue Ebenen hinzugefügt. Die Unterschiede zwischen den Schaltflächen 3 bis 6 werden im Abschnitt Die vier verschiedenen Arten von Ebenen erläutert. Jede neue Ebene wird zunächst in den Vordergrund gestellt. Soll die neue Ebene weiter in den Hintergrund, so muss diese mit der Schaltfläche 1 "Verschiebe die Ebene nach hinten" verschoben werden.

Ebene löschen

Eine Ebene wird gelöscht, indem diese zunächst angeklickt wird und dann auf das Mülltonnensymbol (7) geklickt wird.

Eigenschaften von Ebenen

Eigenschaften von Ebenen können durch einen Doppelklick auf die entsprechene Ebene im Fensterausschnitt rechts unten geändert werden. So kann z.B. der Name der Ebene geändert werden. Je nach Art der Ebene, z.B. Rasterebene, können weitere Eigenschaften geändert werden.

Ebene ausblenden

Mit Anklicken des Augensymbols (9) einer Ebene kann diese aus- und wieder eingeblendet werden.
