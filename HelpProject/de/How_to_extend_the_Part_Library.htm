Erweiterung der Teilebibliothek

Eine besondere Stärke von BlueBrick ist die einfache Möglichkeit, zusätzliche Teile in die Teilebibliothek aufzunehmen. Beim Start von BlueBrick wird die Teileliste anhand der auf der Festplatte vorhandenen Dateien im Ordner "parts" erstellt. Hinzufügen oder Löschen von Teilen ist daher so einfach wie das Hinzufügen oder Löschen von Dateien in diesem Ordner. Um neue Teile zu erstellen und zur Teilebibliothek hinzuzufügen, sollten folgende Schritte durchgeführt werden:

  1. Aufbau des Ordners "parts" vergegenwärtigen
  2. Bild des Teils erstellen unter Beachtung der Dateinamen und -formate für Teile- und Setbilder
  3. Gegebenenfalls eine xml-Datei zur Teilebeschreibung hinzufügen
  4. Für die Erstellung völlig neuer Teile Nutzung der Funktion zum erneuten Laden der Teilebibliothek (empfohlen)
  5. Falls die Kompatibilät des neuen Teils mit LDRAW gewünscht ist: LDRAW Konvertierungsdatei anpassen
  6. Sofern die Kompatibilität mit Track Designer gewünscht ist: Track Designer Konvertierungsdatei anpassen
