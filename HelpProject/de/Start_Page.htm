BlueBrick
 Version 1.5.1.0

 by Alban Nanty

BlueBrick ist eine von Alban Nanty entwickelte, freie Open-Source-Software für Windows/Vista. Diese Software darf ausschließlich privat und nicht kommerziell genutzt werden. Für dieses Programm wird keinerlei Garantie oder Gewährleistung übernommen.

Diese Software wurde besonders für AFOLs zur Vorbereitung von LEGO®-Ausstellungen entwickelt. BlueBrick ist kompatibel mit LDRAW und Train Depot Track Designer. BlueBrick wurde so gestaltet, dass die Bibliothek der in BlueBrick verwendeten Teile einfach erweitert werden kann. Die Ebenen-Funktion erlaubt die übersichtliche Gliederung von Gleisplänen, in speziellen Ebenen können Kommentare eingefügt und besondere Flächen gekennzeichnet werden.
